# Java Tic-Tac-Toe (Console-Based)

A clean, object-oriented implementation of the classic Tic-Tac-Toe game written in Java. This project serves as a foundational demonstration of Java syntax, strict encapsulation, and clean console I/O management.

## 🧠 Key Highlights & Concepts Covered

* **Object-Oriented Design**: Built using rigid OOP principles, ensuring separation of concerns between game logic, board state, and player inputs.
* **Encapsulation**: Strict use of access modifiers (`private`) and getters/setters to protect the board state from direct external modification.
* **Input Validation & Exception Handling**: Implements robust error checking to handle invalid matrix coordinates, preventing crashes from user errors.
* **Multidimensional Arrays**: Leverages a 2D array structure (`char[][]`) to efficiently represent and update the grid matrix.

## 📂 Project Structure
* `Main.java` - Manages the core game loop, turn switches, win/tie conditions, and console rendering.

## 🔧 How To Run
1. Make sure you have **JDK** installed.
2. Compile the main file: `javac Main.java`
3. Run the application: `java Main`
